### 关键特性

* 600MHz XuanTie C906 + 400MHz HiFi5 DSP + 240MHz Cortex-M33
* 1MB SRAM + 16MB PSRAM + 16MB Flash
* Wi-Fi + Bluetooth
* Audio CODEC
* QFN80(8*8mm 0.35pitch)
